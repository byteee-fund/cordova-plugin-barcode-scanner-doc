# Error Codes

| Error Code | Description                  |
| ---------- | ---------------------------- |
| 1001       | API not initialized           |
| 1002       | Not installed                 |
| 1003       | Failed to send request        |
| 1004       | Invalid parameters            |
| 1005       | Thread error                  |
| 1006       | IO error (failed to read image or file) |
| 1007       | Unsupported API               |
